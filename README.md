# WiseAdvisor

**Smart Planning, Secure Legacy.**

WiseAdvisor is a modern, high-performance platform for professional financial and insurance advice. Built with a focus on speed, security, and content delivery.

## Tech Stack

- **Frontend:** Astro (Static Site Generation)
- **CMS:** Sanity.io (Headless CMS)
- **Hosting & Edge:** Cloudflare Pages & Workers
- **Database:** Cloudflare D1
